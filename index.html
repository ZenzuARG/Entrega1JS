Simulador de Cajero Automático

Este proyecto simula un cajero automático utilizando JavaScript puro.

Permite al usuario ingresar mediante un PIN y realizar operaciones básicas como:

  • Consultar saldo
  • Retirar dinero
  • Depositar dinero

Toda la interacción se desarrolla mediante cuadros de diálogo (prompt, confirm, alert) y en la consola del navegador.

El diseño fue estilizado mínimamente para mejorar la presentación del proyecto.
